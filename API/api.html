Each function has an example of the appropriate JSON input, a description of the purpose of the function, and a break down of the inputs.

function move(sprite, dur, dist, height, rotateClockwise, degrees) { if(!rotateClockwise) { degrees = -degrees; } sprite.animate({rotation: sprite.targetRotate+degrees , y: sprite.targetY - height, x: sprite.targetX + dist}, {duration:dur, easing:"ease-in-sine", queue: "b", callback:function () { } }); sprite.targetY -= height; sprite.targetX += dist; sprite.targetRotate += degrees; }

Generic Move

function move(sprite, duration, distance, height, rotate, degree)

A generic method for moving how you want a sprite to move- can go vertical, horizontal, and even spin.

Key Value Description
sprite Sprite The sprite object to be animated
duration int The time in milliseconds the animation will take to execute
distance int The X value for movement
height int The Y value for movement
rotate boolean True- clockwise, False- counterclockwise
degree int How many degrees do you want to rotate? 0 for none

wrapHTMLCanvasToOcanvas

oCanvas wrapHTMLCanvasToOcanvas(HTML5canvasID, background)

Wraps an instance of Ocanvas around a HTML 5 canvas. You need to do this in order to create sprites. (see addSprite)

Key Value Description
HTML5canvasID string You can get this by calling document.getElementById("myCanvas") where "myCanvas is the ID of your HTML5 canvas.
background string sets the canvas background to the hex value (#FFFFFF is a white background).
RETURNS oCanvas Returns the oCanvas object. you need this when creating a sprite.

addSprite

Sprite addSprite(theOCanvas, image, width, height, xcoord, ycoord)

Adds a sprite onto the ocanvas!

Key Value Description
theOCanvas string This is returned from wrapHTMLCanvasToOcanvas(HTML5canvasID, background)
image string The image path to the image for the sprite as a string.
Width int The width for the sprite.
height int The height for the sprite.
xcoord int The x coordinate for the sprite.
ycoord int The y coordinate for the sprite.
RETURNS Sprite Returns a sprite object! you can now call animations with this object and that sprite will be animated!
