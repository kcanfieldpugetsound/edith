1) Download mamp/wamp/lamp (mac, windows, linux)
2) Open Application
3) open the folder ‘htdocs’ (located in the applications folder of your respective amp program.)
4) put the database files (our provided files) inside of htdocs folder
5) start servers
6) go to “PHPMyAdmin” tab on the localhost webpage
7) look for “edith” button on left side
8) click it and rejoice in your awesome skills and setting up this database
